# Titanic-EDA
# 🚢 Titanic Dataset - Simple EDA Project

A simple Exploratory Data Analysis (EDA) project on the Titanic dataset to understand survival patterns of passengers using Python visualization libraries.

## 🧠 Description
This project analyzes the Titanic dataset to find patterns in survival based on factors like gender, age, passenger class, and fare. It uses basic data cleaning and visualization techniques.

## 📊 What This Project Does
- Loads Titanic dataset from Seaborn
- Checks missing values
- Fills missing age values
- Visualizes survival patterns using graphs
- Extracts simple insights from data

## 🛠️ Technologies Used
- Python 3
- Pandas
- Seaborn
- Matplotlib
